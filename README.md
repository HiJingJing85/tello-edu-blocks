# Tello EDU 블록코딩

Tello EDU 드론을 블록코딩으로 쉽게 조종할 수 있는 교육용 프로그램입니다.
Tello SDK 2.0을 지원하며, Python 설치 없이 실행파일 하나로 사용할 수 있습니다.

---

## 주요 기능

- **블록코딩 인터페이스** — 드래그 앤 드롭으로 드론 비행 프로그램을 만들 수 있습니다
- **8개 블록 카테고리** — 기본 제어, 이동, 회전, 플립, 속도, 고급 이동, 미션패드, 흐름 제어
- **미션패드 지원** — Tello EDU 전용 미션패드 감지 및 좌표 기반 비행
- **시뮬레이션 모드** — 드론 없이도 명령어 흐름을 테스트할 수 있습니다
- **실시간 상태 모니터링** — 배터리, 높이, 온도, 비행시간 확인
- **프로그램 저장/불러오기** — JSON 파일로 저장하여 다음에 이어서 사용
- **명령어 보기** — 블록이 어떤 SDK 명령어로 변환되는지 확인 가능
- **단일 실행파일 배포** — Python 설치 불필요

---

## 블록 카테고리

| 카테고리 | 설명 | 포함 블록 |
|---------|------|---------|
| 기본 제어 | 드론 이착륙 | 이륙하기, 착륙하기, 제자리 멈추기, 긴급 정지 |
| 이동 | 방향별 이동 | 위/아래/좌/우/앞/뒤 이동 (20~500cm) |
| 회전 | 좌우 회전 | 시계/반시계 방향 회전 (1~360도) |
| 플립 | 공중 뒤집기 | 왼쪽/오른쪽/앞/뒤 뒤집기 |
| 속도 | 비행 속도 설정 | 속도 설정 (10~100 cm/s) |
| 고급 이동 | 좌표/곡선 비행 | 좌표 이동(go), 곡선 비행(curve) |
| 미션패드 | 미션패드 기능 | 감지 ON/OFF, 감지 방향, 패드 기준 이동, 점프 |
| 흐름 제어 | 프로그램 흐름 | 기다리기, 반복하기 |

## 사용 방법

### 1단계: Tello EDU 전원 켜기

드론 옆면의 전원 버튼을 눌러 Tello EDU를 켭니다.
LED가 깜빡이면 준비 완료입니다.

### 2단계: WiFi 연결

PC에서 WiFi 목록을 열고 **TELLO-XXXXXX** 형태의 네트워크에 연결합니다.
(비밀번호 없음, 인터넷 연결 안 됨은 정상입니다)

### 3단계: 프로그램 실행

- **실행파일 사용 시**: `TelloEDU.exe` (Windows) 또는 `TelloEDU` (macOS)를 더블클릭합니다
- **소스코드 사용 시**: 터미널에서 `python app.py` 를 실행합니다

브라우저가 자동으로 열리며 블록코딩 화면이 나타납니다.

### 4단계: 드론 연결

화면 오른쪽 상단의 **연결** 버튼을 클릭합니다.
"연결됨" 표시가 나타나면 드론과 통신이 시작된 것입니다.

### 5단계: 블록 프로그래밍

1. 왼쪽 카테고리에서 원하는 블록을 선택합니다
2. 블록을 워크스페이스로 드래그합니다
3. 블록끼리 연결하여 비행 순서를 만듭니다
4. 숫자 값을 클릭하여 거리, 각도 등을 변경합니다

### 6단계: 실행

**실행** 버튼을 누르면 블록 순서대로 드론이 움직입니다.

> **안전 주의**: 실행 전 주변에 충분한 공간이 있는지 확인하세요.
> 위험 시 **긴급정지** 버튼을 누르면 모터가 즉시 멈춥니다.

## 기타 기능

### 시뮬레이션 모드
드론을 연결하지 않은 상태에서 실행 버튼을 누르면 시뮬레이션 모드로 동작합니다.
실행 로그에서 명령어 흐름을 미리 확인할 수 있어 수업 중 사전 테스트에 유용합니다.

### 명령어 보기
상단 **명령어 보기** 탭을 클릭하면 블록이 변환된 Tello SDK 명령어를 확인할 수 있습니다.
텍스트 코딩과 연계한 학습에 활용할 수 있습니다.

### 프로그램 저장/불러오기
- **저장**: 오른쪽 저장 버튼을 클릭하면 `.json` 파일로 다운로드됩니다
- **불러오기**: 불러오기 버튼으로 이전에 저장한 파일을 열 수 있습니다

## Tello SDK 2.0 지원 명령어

이 프로그램은 Tello SDK 2.0의 다음 명령어를 지원합니다.

### 제어 명령어
`command`, `takeoff`, `land`, `streamon`, `streamoff`, `emergency`, `stop`

### 이동 명령어
`up`, `down`, `left`, `right`, `forward`, `back`, `go`, `curve`

### 회전 명령어
`cw`, `ccw`

### 플립 명령어
`flip`

### 설정 명령어
`speed`, `rc`, `mon`, `moff`, `mdirection`

### 미션패드 명령어
`go ... mid`, `curve ... mid`, `jump`

### 읽기 명령어
`speed?`, `battery?`, `time?`, `wifi?`, `sdk?`, `sn?`

## 시스템 요구사항

- **OS**: Windows 10 이상 / macOS 10.15 이상
- **브라우저**: Chrome, Edge, Safari (자동으로 열림)
- **드론**: Tello EDU (Tello SDK 2.0 지원 모델)
- **네트워크**: Tello EDU WiFi 직접 연결

## 소스코드로 실행하기

Python 3.8 이상이 설치되어 있어야 합니다.

```bash
# 의존성 설치
pip install flask

# 실행
python app.py
```

브라우저에서 `http://127.0.0.1:5050` 으로 접속됩니다.

## 프로젝트 구조

```
sw_tello/
├── app.py              # Flask 웹 서버 (API + 드론 통신 중계)
├── tello.py            # Tello SDK UDP 통신 모듈
├── build.spec          # PyInstaller 빌드 설정
├── requirements.txt    # Python 의존성
├── templates/
│   └── index.html      # 블록코딩 메인 페이지
└── static/
    ├── css/style.css   # UI 스타일
    ├── js/
    │   ├── blocks.js   # Tello 커스텀 블록 정의
    │   ├── generators.js # 블록 → SDK 명령어 변환
    │   └── app.js      # 프론트엔드 로직
    └── lib/            # Blockly 라이브러리 (로컬 번들)
```
